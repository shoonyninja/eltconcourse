  1. Achievement test
    1.   A test designed to measure general language ability
    2.   A test designed to apportion learners to learning groups
    3.   A test of how much has been learned
    4.   A test of communicative ability
  2. Adjacency pair
    1.   Apologising and offering help
    2.   Offering and accepting help
    3.   Greeting and leave taking
    4.   Introducing oneself
  3. Affective filter
    1.   A hindrance to learning caused by poor planning
    2.   A hindrance to teaching caused by too much stress
    3.   A hindrance to learning caused by negative emotional responses
    4.   A filter which aids learning focus
  4. Aural
    1.   To do with hearing and listening
    2.   To do with speaking and understanding
    3.   To do with speaking
    4.   To do with communication in real time
  5. Cognition
    1.   Remembering, recognising, inferencing, deducing, classifying
    2.   Responding to oral clues and cues
    3.   Learning by imitation and repetition
    4.   Reacting emotionally to the content of a lesson
  6. Context
    1.   The social setting in which language is used
    2.   The words immediately before or following an item of vocabulary
    3.   Where in a text a topic sentence comes
    4.   The theme of a lesson
  7. Deductive learning
    1.   Proceeding from the example to the rule
    2.   Using a rule to produce accurate language forms
    3.   Learning by discovery
    4.   A testing approach to teaching
  8. Concept-checking question
    1.   A form of communicative elicitation
    2.   A question concerned with testing understanding
    3.   A question requiring a relevant personal response
    4.   A question about an idea
  9. FLA
    1.   First Language Acquisition
    2.   Foreign Language Acquisition
    3.   First Language Appropriacy
    4.   First Learning Activity
  10. Inductive learning
    1.   Proceeding from the examples to the formulation of a rule
    2.   Using a rule to form accurate language structures
    3.   Inferring the meaning of unknown words by looking at the co-text
    4.   Using a model sentence as a target to imitate
  11. Interaction
    1.   The use of language to maintain social relationships
    2.   The use of language to communicate emotional states
    3.   The use of language to achieve social services
    4.   The use of language to get what you want
  12. Phonemic analysis
    1.   The study of the sounds of a specific language
    2.   Transcribing English sounds
    3.   The study of sound patterns
    4.   The study of the connection between spelling and pronunciation
  13. PPP
    1.   Presentation - Practice - Production
    2.   Practice - Production - Presentation
    3.   Production - Practice - Presentation
    4.   Practice - Presentation - Production
  14. TTT
    1.   Test - Teach - Test
    2.   Teach - Test - Test again
    3.   Teach - Talk - Test
    4.   Teacher Testing Times
  15. Style
    1.   Variations in formality
    2.   Appropriate language to maintain politeness
    3.   Accent and dialect
    4.   The language of a professional context
  16. Prescriptive grammar
    1.   A grammar which sets out the rules of speaking
    2.   A grammar which seeks to say what is correct
    3.   A grammar which explains how structures express meanings
    4.   A simplified grammar for use with learners
  17. Redundancy
    1.   Unnecessary language when talking too much
    2.   Giving the same information more than once
    3.   Background noise when listening
    4.   Irrelevant details
  18. Skimming
    1.   Reading quickly to locate specific information
    2.   Reading quickly to understand the kind of text it is
    3.   Reading for pleasure without understanding all the words
    4.   Reading to find a single fact in a text
  19. Notions
    1.   Ideas about language
    2.   Concepts such as length, availability, usefulness etc.
    3.   Ideas about the rules of the grammar
    4.   Getting things done in language
  20. Discourse analysis
    1.   An investigation of interactions rather than transactions
    2.   Looking at language structures in sentences
    3.   Looking at language structures in longer texts
    4.   Analysing speech
